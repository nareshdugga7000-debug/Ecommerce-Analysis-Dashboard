# Ecommerce-Analysis-Dashboard
A data visualization dashboard built using Power BI,SQL and MS Excel.

Project Overview

This project focuses on ecommerce data analysis and dashboard creation using Excel, SQL, and Power BI. Multiple datasets were collected and processed to create a single integrated dataset for business insights and visualization.

Tools Used

- Excel
- MySQL / SQL
- Power BI

Project Workflow

1. Created and organized 3 Excel sheets using multiple ecommerce datasets.
2. Imported the datasets into SQL.
3. Performed data cleaning and transformation using SQL queries.
4. Joined multiple tables in SQL to create one final combined table.
5. Imported the final table into Power BI.
6. Built an interactive ecommerce dashboard for data visualization and business insights.

Key Features

- Data Cleaning & Transformation
- SQL Joins
- Interactive Power BI Dashboard
- Ecommerce Sales Insights
- Multi-source Data Integration

Files Included

- Excel Dataset Files
- SQL Query File
- Power BI Dashboard File
- Dashboard Screenshots

Outcome

This project demonstrates practical skills in data analysis, SQL data manipulation, and Power BI dashboard development for ecommerce business insights.
